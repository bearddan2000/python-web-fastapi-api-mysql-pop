# python-web-fastapi-api-mysql-pop

## Description
Simple web app that serves api
for a fastapi project.

Uses sqlalchemy query a table `pop`.

## Tech stack
- python
  - fastapi
  - uvicorn
  - sqlalchemy
- bootstrap
- jquery
- dataTable
- mariadb

## Docker stack
- python:latest
- mariadb:latest

## To run
`sudo ./install.sh -u`
- [Availble at](http://localhost/pop)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
